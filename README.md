# Floral__Fantasy
A fully responsive E-Commerce website for flower shopping built using HTML, CSS, and Bootstrap. This project provides a seamless and elegant shopping experience, showcasing various flower collections, bouquets, and gift options. The design is modern, user-friendly, and optimized for both desktop and mobile platforms.
HTML5: For the structure and content.
CSS3: For styling and animations.
Bootstrap: For layout and responsive design.
